  - 문제: [재구매가 일어난 상품과 회원 리스트 구하기](https://school.programmers.co.kr/learn/courses/30/lessons/131536)

- 결과 캡처(PNG)보다 실제 SQL 텍스트 답안을 남길 것
- Todo 스키마는 제약조건과 인덱스를 명시할 것
